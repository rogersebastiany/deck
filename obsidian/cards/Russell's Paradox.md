## Card Text

> **[Continuous Trap]**
> When your opponent activates a card effect that targets a monster that opponent controls, negate that effect and destroy all face-up monsters with that same name on the field.

## Artwork Description
An elderly philosopher (resembling Bertrand Russell) holding a sign reading "The set of all sets that do not contain themselves." Swirling red vortex with text "Must contain itself" repeating in a loop. SQL and Python code snippets visible in the background referencing self-containment logic.

## Synergies
- Devastating against decks that run multiple copies of the same monster
- Continuous presence pairs with [[The Law of Excluded Middle]] for lockdown
- Philosophical trap chain with [[Kant's Categorical Imperative]]
